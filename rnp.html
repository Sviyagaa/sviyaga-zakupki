Рука на пульсе

Жми канал → метрику → неделю, чтобы дойти до конкретного дня. Видно динамику и точные цифры план/факт на каждый день. Закрытые дни окрашены по выполнению дневного плана — это и есть основа решений на планёрке.
💡 Как считаются дни: план месяца раскладывается по неделям (неделя = 7 дней из 30) и по дням (план месяца ÷ 30). Факт по дню тянется из WB/Ozon API. Сегодня 2 июня — закрыт только день 1, поэтому динамика пока на старте.
Прототип v4 на реальных данных РНП (июнь). Факт есть по WB Свияга (день 1 = 545 938 ₽ заказов при дневном плане ~537 635 ₽ по каналу). Остальные каналы и дни подтянутся из API ежедневно. Дневной план = план месяца ÷ дни; в боевой версии можно задать неравномерный план по дням недели.
Цвет дня: зелёный — дневной план выполнен, жёлтый — близко, красный — недобор (для ДРР наоборот). Будущие дни — бледные (только план).
